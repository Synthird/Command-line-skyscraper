# Command line skyscraper
Using dots to print a skyscraper in the command line.

## Example of a skyscraper that can be printed

This skyscraper has seven floors and a lightning rod that is three dots tall.

```
   .
   .
   .
.......
.......
.......
.......
.......
.......
.......
```
